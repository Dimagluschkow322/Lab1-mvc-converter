## Архитектура MVC (Model-View-Controller)


### Model (Модель)

Класс `ConverterModel` — расположен в пакете `model`:
1. Хранит коэффициенты конвертации (проценты, коэффициенты, дроби)
2. Содержит метод `convert(double value, String from, String to)`
3. Полностью не зависит от интерфейса

### View (Представление)

FXML-файл `ConverterView.fxml` — расположен в `resources`:
1. `TextField` для ввода значения
2. `ComboBox` для выбора типа конвертации
3. `Button` для выполнения конвертации
4. `Label` для вывода результата
5. Макет на основе `VBox` и `HBox`
6. Стилизация: градиентный фон, шрифты, отступы

### Controller (Контроллер)

Класс `ConverterController` — расположен в пакете `controller`:
1. Реализует интерфейс `Initializable`
2. Связывает View (через аннотации `@FXML`) и Model
3. Обрабатывает нажатие кнопки конвертации
4. Обновляет интерфейс по результатам вычислений


## Особенности реализации

1. Полное разделение на слои Model, View, Controller
2. Графический интерфейс на JavaFX с использованием FXML
3. Класс `InputValidator` для валидации пользовательского ввода:
4. Проверка на пустую строку
5. Проверка на возможность преобразования в число
6. Проверка на положительное значение
7. Проверка на превышение максимально допустимого значения
8. Тестирование логики конвертации в методе `main` класса `Main`
9. Стилизация интерфейса через CSS (градиенты, шрифты, отступы)
